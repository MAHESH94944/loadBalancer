      Algorithms Used

      Las Vegas Algorithm

      The Las Vegas algorithm guarantees correct results by randomly selecting servers and assigning tasks to the least loaded one. It runs multiple attempts to ensure it finds a server that meets the load condition, ensuring optimal task assignment.

      Example: If there are three servers with task counts of 1, 0, and 3 respectively, the algorithm may randomly check the server loads multiple times. It will assign a task to the server with 0 tasks, guaranteeing efficient load distribution.

      Monte Carlo Algorithm

      The Monte Carlo algorithm uses randomness to select servers and assigns tasks based on which server has the lowest load. It runs a number of iterations to enhance the probability of making the optimal choice.

      Example: Given three servers with loads of 2, 3, and 1, the algorithm randomly samples servers 10 times, comparing their loads. It assigns the task to the server with 1 load if it is sampled during its iterations.
